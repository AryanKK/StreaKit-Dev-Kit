# Playground

::: warning Personal project in development

This playground ships with a **personal project in active development**. Treat it as a demo harness, not a production reference.

:::

The playground is a live app that uses `@streakit/core` so developers can test a full integration.

## Run locally

```bash
pnpm dev:demo
```

By default, the playground runs at:

- `http://localhost:5173`

## What it demonstrates

- Creating a streak
- Recording activity
- Freezing and unfreezing
- Reading computed status values
- Visualizing progress to next milestone

Use this app as a starting template for your own onboarding sample or internal QA harness.
